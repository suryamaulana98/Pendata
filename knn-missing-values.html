1. K-Nearest Neighbor (KNN)#

K-Nearest Neighbor (KNN) merupakan salah satu algoritma dalam machine learning yang digunakan untuk melakukan klasifikasi maupun regresi.

Algoritma ini bekerja berdasarkan prinsip kedekatan antar data. Suatu data baru akan diprediksi nilainya dengan melihat data lain yang memiliki karakteristik paling mirip.

Konsep utama KNN adalah:

  • Data yang mirip cenderung memiliki nilai atau kelas yang mirip.

  • Tingkat kemiripan antar data diukur menggunakan jarak (distance).

Huruf K dalam KNN menunjukkan jumlah tetangga terdekat yang digunakan dalam proses pengambilan keputusan.

2. Cara Kerja Algoritma KNN#

Langkah kerja algoritma KNN secara umum adalah:

  1. Menentukan nilai K (jumlah tetangga terdekat).

  2. Menghitung jarak antara data yang akan diprediksi dengan seluruh data dalam dataset.

  3. Mengurutkan jarak dari yang paling kecil.

  4. Memilih K data dengan jarak paling kecil.

  5. Menentukan hasil prediksi:

  • Pada klasifikasi, digunakan metode voting.

  • Pada regresi, digunakan rata-rata nilai dari K tetangga.

3. Euclidean Distance#

Salah satu metode yang paling umum digunakan untuk menghitung jarak dalam KNN adalah Euclidean Distance.

Euclidean Distance menghitung jarak garis lurus antara dua titik dalam ruang multidimensi.

Rumus Euclidean Distance#

\[ d(p,q) = \sqrt{\sum_{i=1}^{n}(p_i - q_i)^2} \]

Keterangan:

  • \(d(p,q)\) = jarak antara data \(p\) dan \(q\)

  • \(p_i\) = nilai atribut ke-i pada data \(p\)

  • \(q_i\) = nilai atribut ke-i pada data \(q\)

  • \(n\) = jumlah atribut

Semakin kecil nilai jarak, maka semakin tinggi tingkat kemiripan antara dua data.

4. Missing Value#

Missing value adalah kondisi dimana suatu atribut dalam dataset tidak memiliki nilai.

Missing value dapat disebabkan oleh beberapa faktor:

  • kesalahan pencatatan data

  • data tidak lengkap

  • kesalahan proses pengumpulan data

Jika tidak ditangani dengan baik, missing value dapat mempengaruhi hasil analisis dan performa model.

5. KNN Imputation#

KNN Imputation adalah metode untuk mengisi missing value menggunakan konsep KNN.

Langkah-langkahnya:

  1. Mengidentifikasi data yang memiliki missing value.

  2. Menghitung jarak antara data tersebut dengan data lain.

  3. Mengurutkan jarak dari yang terkecil.

  4. Memilih K tetangga terdekat.

  5. Menghitung nilai estimasi menggunakan rata-rata nilai dari tetangga terdekat.

Contoh Perhitungan Euclidean Distance#

Perhitungan jarak antara rumah R5 dan R1.

\[ d = \sqrt{(140-120)^2 + (100-90)^2 + (3-3)^2} \]
\[ d = \sqrt{400 + 100 + 0} \]
\[ d = \sqrt{500} = 22.36 \]

Perhitungan yang sama dilakukan terhadap semua data rumah lainnya untuk menemukan tetangga yang paling mirip.

import pandas as pd

data = {
    'Rumah':['R1','R2','R3','R4'],
    'Luas_Tanah':[120,200,150,90],
    'Luas_Bangunan':[90,150,110,70],
    'Jumlah_Kamar':[3,4,3,2],
    'Harga':[850,1500,1000,600]
}

df = pd.DataFrame(data)
df

import numpy as np

target = np.array([140,100,3])

features = df[['Luas_Tanah','Luas_Bangunan','Jumlah_Kamar']].values

distances = np.sqrt(((features-target)**2).sum(axis=1))

df['Distance'] = distances

df.sort_values('Distance')

Kesimpulan#

Dengan menggunakan metode K-Nearest Neighbor, nilai harga rumah R5 dapat diperkirakan berdasarkan rumah yang memiliki karakteristik paling mirip.

Metode ini bekerja dengan mencari tetangga terdekat berdasarkan Euclidean Distance kemudian menggunakan rata-rata nilai dari tetangga tersebut.

KNN merupakan metode yang sederhana namun cukup efektif untuk:

  • klasifikasi

  • regresi

  • imputasi missing value
